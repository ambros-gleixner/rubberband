How do I interpret the comparison view?

The compare view shows the same data as the Result Summary view, but for multiple test runs. The color of the cell represents how large the spread of values is ((largest-smallest)/base) for that particular instance and metric. Green means that the baserun is better than the comparison, red means that it is worse. In a comparison view with more than two testruns red indicates the value is better (worse) than the best (worst) of the comparisons.
