## HTTP Requests

An HTTP Request consists of `method url body`

The possible methods are:
    - GET
    - POST
    - PUT
    - DELETE
    - UPDATE

The URL has the following form:
`rubberband.example/path/to/somewhere/file.ext?arg1=val1,arg2=val2#hook`
